# TikTok Clone

基于 Next.js 14 构建的抖音克隆项目。

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide Icons

## 功能特点

- 响应式布局
- 视频流展示
- 直播列表
- 搜索功能
- 个人中心
- 登录注册

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
